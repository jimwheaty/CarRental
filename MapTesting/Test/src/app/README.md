
# Δείγμα Angular web client με openlayers

## ..Προαπαιτούμενα:
* [NodeJS]
* [Angular-cli]
* [Typescript]

## ..Στήσιμο περιβάλλοντος

```bash
ng new poi-cli
cd poi-cli
npm i openlayers
npm i @types/openlayers -D
```
Χρησιμοποιείτε τον editor της επιλογής σας

## ..debugging

```bash
ng serve
```
 http://127.0.0.1:4200


## ..deploy

```bash
npm run build
```
